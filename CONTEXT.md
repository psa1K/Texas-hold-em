# Texas Hold'em — 前端展示域

全栈德州扑克应用中，与牌桌 UI 和玩家本地偏好相关的展示概念。

## Language

**牌组皮肤（Card Deck Skin）**:
一套完整的美术素材（52 张牌面 + 牌背），通过目录与文件命名规则与游戏逻辑中的「牌」对应。
_Avoid_: 牌面（易与单张翻开展示混淆）、主题（过于宽泛）

**牌串（Card String）**:
服务器下发的牌标识，如 `A♠`、`T♥`；由游戏引擎定义，与皮肤无关。
_Avoid_: 牌面路径、文件名

**皮肤注册表（Deck Registry）**:
前端维护的可用牌组列表；每项包含目录 id、显示名称、图片后缀等元数据，供循环切换与后续扩展。
_Avoid_: 硬编码路径

**皮肤偏好（Deck Preference）**:
玩家选择的牌组皮肤 id，保存在浏览器 localStorage（键 `thp_card_deck`），刷新后仍生效。
_Avoid_: 用户设置（与 LLM 服务器配置混淆）
